# Online-Rechner

!!! tip "Gemeinsam wachsendes Werkzeug"
    Dieser Bereich füllt sich im Lauf des Kurses. Nachdem ihr einen Berechnungstyp **per Hand sicher beherrscht**, ergänzen wir gemeinsam einen digitalen Rechner zur Ergebniskontrolle.

## Grundsatz

Die Handrechnung ist und bleibt **der Lerninhalt**: Sie wird in der Technikerprüfung verlangt, Sie schult das Verständnis und Sie macht euch unabhängig vom richtigen Ergebnis eines Tools.

Der Online-Rechner ist **Werkzeug für die Praxis**: Schnelle Variantenrechnung in der Beratung, Ergebniskontrolle, Sensitivitätsanalysen.
